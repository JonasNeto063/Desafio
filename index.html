Tag h1

A tag (Heading 1) é o principal elemento de cabeçalho HTML, utilizado para definir o título mais importante de uma página web. Ela possui valor semântico e estrutural, indicando aos navegadores e mecanismos de busca (como o Google) o assunto central do documento.

Tag h2

A tag em HTML define um cabeçalho (ou título) de segundo nível. Ela é usada para criar subtítulos e organizar o conteúdo de uma página em seções menores, melhorando a leitura e ajudando motores de busca como o Google a entenderem a estrutura do seu site.

Tag h3

A tag em HTML representa um título de terceiro nível (ou heading 3).

Ela é usada para organizar a estrutura do conteúdo de uma página web, indicando que o texto em questão é um subtópico de um título.

Tag h4

Assim como o, ela serve para estruturar o conteúdo de uma página web, mas com um nível ainda menor de importância (e geralmente um tamanho de fonte menor por padrão). Ela é usada para criar subseções dentro de um bloco que já está sob um título.

Tag h5

Seguindo a lógica das tags anteriores, ela serve para subdividir o conteúdo que está dentro de uma seção. Ela já está quase no final da escala de títulos do HTML (que vai do ao).

Tag h6

Na prática, é uma tag muito pouco utilizada no dia a dia, pois é raro um texto ou site precisar de seis níveis de profundidade em seu índice. Ela costuma aparecer apenas em documentações técnicas extremamente detalhadas ou dicionários.

Tag mark

Ela é uma das tags mais fundamentais e utilizadas na web, servindo para delimitar e organizar blocos de texto comum em uma página (como este texto que você está lendo agora).

Tag small

A tag em HTML define textos menores, como comentários laterais, direitos autorais e avisos legais. Ela reduz o tamanho da fonte em um nível em relação ao texto normal

Tag i

A tag no HTML representa um texto formatado em itálico. Ela indica partes do conteúdo que possuem uma "voz", humor ou tom diferente, como palavras em outro idioma, termos técnicos, pensamentos ou nomes de navios.

Tag u

A tag (do inglês paragraph) é o elemento HTML fundamental usado para definir e estruturar parágrafos de texto em uma página web. Ela exige uma tag de abertura e uma de fechamento.

Tag strong

A tag é um elemento HTML que indica forte importância, seriedade ou urgência para o seu conteúdo. Visualmente, os navegadores a renderizam em negrito, mas sua principal função é semântica, informando a relevância do texto para mecanismos de busca e leitores de tela.

Tag ol

A tag (Ordered List) define uma lista ordenada em HTML, criando uma sequência numerada ou alfabética. Os itens dentro dela são representados pela tag (List Item)

Tag ul

A tag (do inglês unordered list) cria listas não ordenadas em HTML, tipicamente exibidas com marcadores (bolinhas). Ela deve ser usada sempre que a ordem dos itens não alterar o significado da lista.

Tag li

Em HTML, a tag significa Item de Lista . Ela é usada para definir um item individual dentro de uma lista e deve sempre ser colocada dentro de uma tag de lista pai.

Tag a

A tag a (de anchor ou âncora) é usada para criar hiperlinks em HTML. Ela permite direcionar o usuário para outras páginas da web, seções internas de um mesmo documento ou arquivos para download.

Tag hr

Usado para quebrar um parágrafo, usando uma linha na horizontal.

Tag sub

É utilizado escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo.

Tag sup

Especifica um texto em linha que deve ser exibido como sobrescrito por razões tipográficas.

Tag blockquote

Usado para citar um texto de outra fonte.
